Cloud Automation Services API Query
This component leverages the unofficial Python SDK For Cloud Automation Services (caspyr) to interact with the Cloud Automation Services API and return details about a Cloud Automation Services org. A user can enter their CSP Refresh Token, and the Angular driven frontend will query the API backend, resolving the data in an ng2-chart!
